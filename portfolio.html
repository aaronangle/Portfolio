Pleasantly

This website was created by myself and three other individuals. A user can enter in any ingredients that they have readily available and our website will provide recipes that they are able to make from the ingredients. Users are also able to search by recipe names to find things that they want to cook. The site makes a call to the Edamam API based on user input to obtain the data that is rendered to the page.

Weather Dashboard

The weather dashboard communicates with the OpenWeather API based on user input to get the forecast for the city that is searched. The site provides the current conditions as well as the forecast for the next five days. Searches are stored in the local storage so users can easily jump between cities that have already been searched for.

Daily Planner

The Daily Planner uses local storage to save user input to the site. Each box on the planner represents an hour of the day. The boxes are color coded based on the current time of the day. The current hour is highlighted in a bright blue, the upcoming hours are highlighted in darker blue, and the hours that have passed are marked by gray.
